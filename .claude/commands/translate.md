argument-hint: "[目标语言，默认英文]"

目标语言 = "$ARGUMENTS"（为空则默认翻译成 English）。

把**当前正在学的这一层教案**（或用户刚指定的那段内容/文件）翻译成目标语言：

- 完整、忠实地翻译，保留 markdown 结构、代码块和命令原样（命令不要翻译）。
- 术语用目标语言里通行的说法（如 "终端" → "terminal"，"权限" → "permissions"）。
- 翻完直接展示给用户；除非用户明确要求，不要写入文件覆盖原中文教案。
- 如果用户没说翻哪部分，就问一句："要翻译当前这一层，还是整个 README？"
